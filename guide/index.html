ChatLine 对话增强插件

在浏览器中使用 AI，它会成为你的效率搭档。对话时间轴、文件夹、提示词、复制数学公式、文本高亮、追问等功能，把效率拉满。

✨ 功能特性

功能介绍

强大的工具，让你的 AI 交互更高效、更有条理。

📅

对话时间线

把你跟 AI 的对话整理成可交互时间轴，跳转对话，收藏对话。

📐

LaTeX 公式复制

智能识别任意网页上的公式，点击即可复制它的 LaTeX 格式。

🖼️

图片去水印

启用后，自动去除 Gemini 生成图片上的水印。

💬

提示词管理

快速打开提示词面板，一键调取常用提示词。

▶️

代码运行器

在网页上运行简单的代码块，支持 JavaScript、Python、JSON、Markdown 等热门语言。

📁

文件夹

跨平台收藏问题。统一管理，轻松访问所有重要对话。

↩️

智能回车

单击回车换行，双击回车发送。增强多行输入的灵活性。

🔗

URL 快速跳转

自动检测纯文本 URL，添加可点击的快捷按钮。

📤

对话导出

一键导出当前对话为 Markdown 或 JSON 格式，归档、分享、二次编辑，对话不再被困在浏览器里。

🔮

敬请期待

更多实用功能正在路上，让你的 AI 体验更上一层楼。

📅 对话时间轴

对话时间轴

  • 在 AI 对话页面右侧自动生成时间轴，每条提问对应一个节点
  • 几十轮对话也能一眼总览，点击节点秒跳，告别疯狂滚轮
📁 文件夹

文件夹

  • 把单条问答或整页对话收藏到文件夹中，支持多层级、拖拽排序、置顶、备注
  • AI 给的好答案、关键代码永久留存，下次轻松找到，再也不丢失在对话海里
💬 提示词

自定义提示词库

  • 把常用提示词保存起来，对话框左侧一键插入
📐 公式

公式复制

  • 点击页面上任何数学公式，直接复制为 LaTeX 或 MathML 格式
  • 写论文、做笔记时不用截图重打，一键复制 LaTeX 或 MathML 格式
🖍️ 高亮

文本高亮

  • 选中网页任意文字进行荧光笔标注，4 种样式 + 15 种颜色 + 想法注释
  • 像读纸质书一样圈重点 + 做笔记，重点信息视觉上一眼锁定
🐌 宠物

电子宠物

  • AI 思考时，输入框上方有蜗牛 / 僵尸 / 蚂蚁 / 巫师在欢快走动
  • 等 AI 回复时有点小乐子，让长时间用 AI 也能保持好心情
📤 导出

对话导出

  • 一键把当前对话导出为 Markdown 或 JSON 文件，方便归档、分享、二次编辑
  • AI 给的优质回答不再被困在浏览器里，沉淀到你自己的笔记、文档、知识库中
☁️ 同步

Google Drive 云同步

  • 一键把所有数据上传 / 下载到你自己的 Google Drive
  • 换电脑、换浏览器数据跟着走，数据存你自己的云盘，插件碰不到
